# ElasticFlow用来解决什么?

数据中台作为其核心问题之一就是数据逻辑分层，那么我们要如何实现由一个数据层的数据流引导入下一层呢？
ElasticFlow就是为解决该问题而生。



业务中我们或许需要：

	原始日志进入数据库层，
	原始的各种类型数据库数据进入仓库
	原始大数据进入机器学习中间数据层
	机器学习数据进入用户展示层
	通用搜索服务
	...
综上任务都可以通过ElasticFlow，使用其可控的可计算流管道来实现。
